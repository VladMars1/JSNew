Команди Git

Нижче наведені основні команди Git:

Команда Опис
git init Ініціалізує Git репозиторій в поточній директорії
git clone url Клонує репозиторій за вказаною URL-адресою
git status Показує поточний стан робочої директорії та індексу
git add file Додає змінені або нові файли до індексу
git commit -m "message" Зберігає зміни до локального репозиторію з повідомленням коміту
git push Завантажує локальні зміни на віддалений репозиторій
git pull Оновлює локальний репозиторій до останньої версії з віддаленого репозиторію
git branch Показує список гілок в репозиторії
git checkout branch Перемикає на вказану гілку
git merge branch Об'єднує вказану гілку з поточною гілкою
